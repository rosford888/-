# Aubade 风格灵感站点

这是一个静态网页原型，用于在中国大陆展示类似 https://int.aubade.com/ 的法式内衣体验。页面包含新品陈列、筛选、门店信息、会员/服务保障与订阅入口，可直接本地预览。

## 使用方式

1. 在项目根目录启动本地服务器（任选其一）：
   - Python: `python -m http.server 8000`
   - Node: `npx serve .`
2. 浏览器访问 `http://localhost:8000` 即可体验筛选与门店切换互动。

页面完全前端实现，无需额外依赖。可根据实际商品与门店数据扩展 `script.js` 中的数组。
